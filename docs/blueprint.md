# **App Name**: OCAA FISI UNMSM Website

## Core Features:

- Hero Section: Display a hero section with the OCAA title, subtitle, and CTAs.
- Carousel: Implement an autoplaying carousel with controls, indicators, pause on hover (desktop), pause on focus (keyboard), and swipe/drag support (mobile).
- Services Section: Display OCAA functions/services using cards.
- Document List: Display a searchable and filterable list of documents with links to local PDFs or external URLs.
- Contact Form: Implement a contact form with basic validation and visual feedback.

## Style Guidelines:

- Primary color: UNMSM wine/red #621518 for accents, buttons, and headings (use sparingly).
- Background: #FAFAFA (clean light look).
- Hover accent tint: #8E282C.
- Font: Inter, sans-serif (Google Fonts). Use for headings and body.
- Spacious layout (plenty of whitespace). Cards with soft shadows and subtle rounded corners.
- Use Framer Motion for subtle fade/slide-in animations on viewport entry. Subtle hover animations for buttons/cards (no exaggerated motion).
